## üöÄ Installation & Setup

### Prerequisites
- Node.js (for Vite development server)
- Modern web browser

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/Rizwan-Ullah-tech/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

## üé® Customization

### Updating Personal Information
1. Replace profile picture in `images/Profile.jpg`
2. Update contact information in the hero section
3. Modify social media links throughout the site
4. Replace resume file (`resume.pdf`)

### Adding New Projects
1. Update the projects section in `index.html`
2. Add project screenshots to `images/` folder
3. Update project links and descriptions
4. Adjust grid layout if needed

### Theme Customization
- Modify Tailwind configuration in the HTML `<script>` tag
- Update color scheme using CSS custom properties
- Adjust animations and transitions
- Change fonts by updating Google Fonts import

### Adding New Experience
- Follow the existing structure in the experience section
- Add appropriate icons and colors
- Update duration and descriptions
- Maintain consistent styling
