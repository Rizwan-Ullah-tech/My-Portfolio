# 🌐 Personal Portfolio

A clean and responsive **frontend-only** personal portfolio website built using **HTML**, **Tailwind CSS**, and **JavaScript**.

This portfolio showcases my skills, projects, contact information, and more. It is designed to be simple, fast, and professional — perfect for job applications, freelancing, or sharing my work online.

---

## 🚀 Features

- Fully responsive layout (mobile-first)
- Hero section with profile picture
- About section with details about me
- Project showcase (with optional screenshots)
- Contact form (currently for display only — no backend integration)
- Simple and organized codebase using HTML + Tailwind CSS + JS

---

## 📁 Project Structure

my-portfolio/
├── index.html # Main HTML file
├── README.md # Project documentation
├── assets/
│ ├── images/ # Profile picture, project screenshots
│ │ └── profile.jpg # Your photo for the hero section
│ └── js/
│ └── app.js # JavaScript file (for interactivity)

---

## 🖥️ How to Use

1. Download or clone the repository:

   ```bash
   git clone https://github.com/Rizwan-Ullah-tech/My-Portfolio.git
   cd My-Portfolio
2. Open index.html in your web browser.

3. Customize the content:

   - Replace the profile picture at assets/images/profile.jpg

   - Edit index.html to update your details, projects, and contact information

   - Add or update screenshots in assets/images/ and reference them in HTML

4. The portfolio is fully responsive and optimized for all screen sizes.

5. You can modify assets/js/app.js for any JavaScript functionality.

## 📸 Adding Your Photos

To personalize your portfolio, add your own images in the assets/images/ folder.

🔹 Profile Picture
File name: profile.jpg

Recommended size: 400x400 pixels

Location: assets/images/profile.jpg

🔹 Project Screenshots (Optional)
File format: project-1.jpg, project-2.jpg, etc.

Recommended size: 800x600 pixels

Store inside assets/images/

⚠️ Optimize (compress) your images for better performance.
